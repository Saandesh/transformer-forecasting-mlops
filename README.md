# Transformer Time-Series Forecasting + MLOps

An end-to-end AI forecasting system using Transformer models for time-series prediction with MLOps integration.

## Project Goal

This project aims to build a production-grade forecasting system capable of:
- Predicting future stock market trends
- Training Transformer-based deep learning models
- Tracking experiments using MLOps tools
- Deploying APIs and dashboards
- Monitoring model performance
